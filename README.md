# Discord-bot-for-used-bikes
GSX-R Hunter is an automated Discord bot built for motorcycle enthusiasts! It monitors Hungarian classifieds 24/7, scraping for 2003-2006 Suzuki GSX-R sportbikes. Powered by Python, BeautifulSoup, and SQLite, it analyzes descriptions to categorize bikes as street-legal or track-only, delivering real-time, categorized alerts right to your server.
